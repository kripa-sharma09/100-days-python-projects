# Day 03 - Treasure Island Game

## Description
A beginner-friendly text-based adventure game built in Python where the player makes choices to find hidden treasure. The game uses conditional statements to create different story paths and endings.

## Concepts Used
- if / elif / else statements
- Nested conditions
- Logical operators
- User input
- String methods
- Control flow

## How to Run
1. Open the project folder
2. Run `main.py`
3. Follow the on-screen choices

## Features
- Multiple choice-based paths
- Different game endings
- Interactive storytelling

## Example Gameplay
Welcome to Treasure Island.
Your mission is to find the treasure.

You're at a crossroad. Where do you want to go?
Type "left" or "right"

## Learning Outcome
This project helped me understand how decision-making works in Python using conditional statements.
