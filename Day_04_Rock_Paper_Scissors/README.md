# Day 04 - Rock Paper Scissors

## Description
A Python implementation of the classic Rock Paper Scissors game where the user plays against the computer.

## Concepts Used
- random module
- Lists
- Conditional statements
- User input
- Game logic

## Features
- Random computer choices
- Win/Lose/Draw conditions
- Interactive gameplay

## Learning Outcome
This project helped me understand how to use randomisation and lists to build interactive games in Python.
